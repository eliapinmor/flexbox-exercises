# FLEXBOX EXERCISES

## Contenido del proyecto
Consta de 3 partes: Flexbox, Grid y Animación. Se trata de usar el css para distribuir los elementos a nuestro gusto y que nos ayude a que la experiencia del usuario sea mejor.
### Flexbox
Aquí he usado display: flex para distribuir elementos, centrar divs de manera vertical y horizontal.
### Grid
En esta parte he usado grid, ordenando de manera uniforme los elementos o desardenandolos a nuestro gusto
### Animación
en este apartado he usado distintos elementos como: ..., .... para hacer animaciones desde el propio css
## ¿Cómo arrancar este proyecto?
Primero clónate este repositorio en tu carpeta `https://github.com/eliapinmor/flexbox-exercises` .
Para hacer este proyecto he usado el editor de código Visual Studio Code, solo será necesario tener la extensión de LiveServer instalada y arrancarla, entones puedes visualizar todas las páginas.
